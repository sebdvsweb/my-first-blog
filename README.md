# 📘 Projet Blog — Code GT

## 🎯 Objectif

Créer ton **premier blog en HTML / CSS avec Bootstrap** à partir du projet fourni.

👉 Tu vas :
- comprendre une structure HTML réelle
- modifier un site existant
- créer tes propres articles

---

## 🚀 Consignes générales

⚠️ **IMPORTANT :**
- Lis **tous les commentaires dans les fichiers HTML**
- Ne supprime rien sans comprendre
- Teste ton site régulièrement dans le navigateur

---

## 🧱 Étape 1 — Prendre en main le projet

1. Ouvre le projet dans ton éditeur de code  
2. Lance le fichier `index.html` dans ton navigateur  
3. Observe :
   - la structure
   - les classes Bootstrap
   - les commentaires

---

## 🧭 Étape 2 — Créer la navbar

👉 Objectif : comprendre et modifier la navigation

À faire :
- compléter ou modifier la **navbar**
- changer :
  - le nom du site
  - les liens (Accueil, Articles, etc.)

💡 Tu peux :
- ajouter un lien vers tes articles
- renommer les menus

---

## ✏️ Étape 3 — Modifier les contenus de démonstration

👉 Le projet contient du contenu "fake"

À faire :
- remplacer :
  - les titres
  - les textes
  - les descriptions

💡 Astuce :
- utilise du texte simple au début
- puis améliore progressivement

---

## 🖼️ Étape 4 — Personnaliser les images

À faire :
- remplacer toutes les images
- utiliser :
  - tes images
  - ou des images libres (Unsplash, etc.)

⚠️ Pense à :
- garder des tailles cohérentes
- vérifier l’affichage

---

## 📝 Étape 5 — Créer ton premier article

👉 Utilise le fichier **example HTML fourni**

À faire :
1. Ouvre le fichier exemple  
2. Comprends sa structure  
3. Crée un nouvel article à partir de ce modèle  

Tu dois modifier :
- le titre  
- la date  
- l’auteur  
- le contenu  
- les images  

💡 Tu peux écrire sur :
- les cours Code GT  
- un sujet perso (jeu vidéo, sport, musique…)  

👉 **aucune contrainte de thème**

---

## 🔁 Étape 6 — Créer un deuxième article

👉 Une fois le premier terminé :

1. Duplique le fichier du premier article  
2. Renomme-le  
3. Modifie complètement le contenu  

⚠️ Important :
- ne laisse aucun contenu identique  
- change :
  - titre  
  - texte  
  - images  

---

## 🧪 Étape 7 — Améliorer ton blog

Tu peux :
- ajouter des articles  
- supprimer des sections inutiles  
- modifier le design  

💡 Idées :
- changer les couleurs (Bootstrap)  
- modifier la disposition (grid)  
- ajouter des images dans les articles  

---

## 📌 Règles importantes

✅ Tu dois :
- lire les commentaires dans le code  
- comprendre ce que tu modifies  
- tester régulièrement  

❌ À éviter :
- copier/coller sans comprendre  
- laisser du contenu de démo  

---

## 🎓 Objectifs pédagogiques

À la fin, tu dois être capable de :
- comprendre une structure HTML complète  
- utiliser Bootstrap (`container`, `row`, `col`)  
- modifier un projet existant  
- créer plusieurs pages liées entre elles  

---

## 🚀 Bonus (facultatif)

Si tu veux aller plus loin :
- ajouter un maximum d'articles
- améliorer le design  
- personnaliser complètement le site
